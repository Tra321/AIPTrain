# Todo Reminder — 待办事项提醒工具 系统介绍

## 系统名称

**Todo Reminder**（j），是一款基于浏览器原生通知能力的轻量级任务管理与定时提醒应用。

## 核心功能模块

1. **任务创建与编辑模块**：支持录入任务名称、备注说明、截止时间（datetime-local 精确到分钟）及优先级（高/中/低三级），提供结构化的待办数据模型，确保每条任务信息完整可追溯。

2. **智能提醒模块**：系统每 15 秒轮询一次所有未完成任务的截止时间，当任务到达设定时间时自动触发浏览器 Notification 推送提醒，并在界面上以"已逾期"状态醒目标识；同时提供"延后 10 分钟"快捷操作，支持灵活推迟提醒。

3. **筛选与检索模块**：内置全部、进行中、已逾期、已完成四种视图切换，配合关键词实时搜索，可快速定位目标任务；列表按完成状态、截止时间、优先级三维排序，确保重要紧急事项始终置顶。

4. **数据统计与持久化模块**：顶部概览面板实时展示全部、即将到期、已逾期、已完成四项统计指标；所有数据通过 localStorage 自动持久化，刷新或关闭页面后数据不丢失。

## 目标用户群体

面向日常工作节奏快、需同时追踪多项任务的职场人士及自由职业者。这类用户的核心需求是：低门槛快速记录、准点提醒不遗漏、一目了然掌握全局进度。系统零注册、零配置，打开即用，契合对效率工具"轻、快、准"的诉求。

## 创新点与竞争优势

- **零依赖提醒链路**：不依赖第三方推送服务或后端服务器，完全基于浏览器 Notification API 实现本地定时提醒，无隐私泄露风险，无需网络连接即可工作。
- **渐进式状态感知**：任务状态从"等待提醒"→"30 分钟内到期"→"已逾期"逐级演进，配合色彩编码（黄色预警、红色逾期、绿色完成），形成视觉上的时间压力梯度，驱动用户及时行动。
- **极简部署架构**：纯前端 React + Vite 构建，无后端依赖，可静态部署至任意托管平台，运维成本趋近于零，同时通过响应式布局适配桌面与移动端。
